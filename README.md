# newsletter

A newsletter made with Mailchimp API, Node and ExpressJs

## Run Locally

Clone the project

```bash
  git clone https://github.com/Ghoes18/newsletter
```

Go to the project directory

```bash
  cd newsletter
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Tech Stack

**Client:** HTML5, CSS3

**Server:** Node, Express, Mailchimp API
